# Image-Compression
Image compression using K means



Language
Python 2.7

Required Libraries:
time
numpy
PIL
argparse



Commandline options:
-image - Name of Image eg:Koala, Penguins (Without the extension .jpg)
-k - K value

Example:
python kmeans.py -image "Koala" -k 2


Note:
Keep Images in the same folder as kmeans.py
Images should be in JPG format
